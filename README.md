# ⚡ ПолитАктив — Платформа кадрового резерва

**ПолитАктив** — это интерактивная веб-платформа для Молодёжного Парламента, которая автоматизирует учёт активности участников, формирует прозрачный рейтинг и помогает кадровой службе находить наиболее перспективных кандидатов.

---

## 📌 О проекте

Платформа предназначена для:

* учёта участия в мероприятиях
* автоматического начисления баллов
* формирования рейтинга участников
* упрощения работы организаторов
* отбора кандидатов в кадровый резерв

Система работает как онлайн-сервис и может также запускаться локально для демонстрации.

---

## 🚀 Основные возможности

### 👤 Для участников

* Просмотр списка мероприятий
* Фильтрация по направлениям (IT, Социальное, Медиа и др.)
* Получение баллов за участие
* Бонусные баллы за призы (стажировки, мерч, гранты)
* Генерация QR-кода для подтверждения участия
* Личный кабинет с прогрессом
* Уровни активности (Новичок → Активист → Лидер → Премиум)
* Общий рейтинг участников

---

### 📋 Для организаторов

* Подтверждение участия через QR-код
* Управление списком участников
* Просмотр истории проведённых мероприятий
* Публичный профиль организатора
* Рейтинг доверия и отзывы

---

### 🔍 Для кадровой службы (Инспектор)

* Умный поиск кандидатов
* Фильтрация
* Просмотр достижений участников
* Быстрое формирование отчёта (PDF / текстовый файл)
* Полное портфолио кандидата

---

## 📦 Установка и запуск

### 1. Установите Node.js

Проверьте установку командой:

```
node -v
```

Если версия отображается — всё готово.

---

### 2. Установите зависимости

Перейдите в папку проекта и выполните:

```
npm install express sqlite3 body-parser cors
```

---

### 3. Запустите сервер

```
node server.js
```

---

### 4. Откройте проект в браузере

```
http://localhost:3000
```

---

## ⚙️ Первый запуск

При первом запуске:

* автоматически создаётся база данных `politaktiv.db`
* добавляется **55 тестовых мероприятий**
* система сразу готова к работе

---

## 📡 Режим Offline

Если сервер выключен:

* фронтенд автоматически переключается
* используются встроенные демонстрационные данные
* интерфейс продолжает работать без ошибок

---

## ⚠️ Важно

* Не используйте русские буквы в названиях папок проекта
* Убедитесь, что `index.html` находится в папке `public`
* База данных создаётся автоматически — вручную её создавать не нужно
